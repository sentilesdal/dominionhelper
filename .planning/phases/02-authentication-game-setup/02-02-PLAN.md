2. `getKingdomCardCount(page: Page): Promise<number>`
   - Count kingdom cards visible in the kingdom viewer
   - Primary approach: `return page.locator('.kingdom-viewer-group .full-card-name.card-name').count()`
   - If that selector doesn't work, fall back to AngularJS scope access:
     ```typescript
     return page.evaluate(() => {
       const el = document.querySelector('.kingdom-viewer-group');
       if (!el || !(window as any).angular) return 0;
       const scope = (window as any).angular.element(el).scope();
       return scope?.cards?.length ?? scope?.kingdom?.length ?? 0;
     });
     ```
   - The function should always return a number (0 if nothing found)

3. `waitForMyTurn(page: Page, playerName: string, timeout?: number): Promise<{ turn: number }>`
   - Default timeout: 30000 (30 seconds)
   - Use `expect.poll()` pattern from Playwright for eventual consistency:
     ```typescript
     let turnNumber = 0;
     await expect(page).toPass(async () => {
       const info = await page.evaluate((name: string) => {
         const log = document.querySelector('.game-log');
         if (!log) return null;
         const text = (log as HTMLElement).innerText;
         const regex = /(?:Turn|Zug|Tour|–•–æ–¥) (\d+) - (.+?)(?:\n|$)/g;
         let match;
         let lastMatch: RegExpExecArray | null = null;
         while ((match = regex.exec(text)) !== null) lastMatch = match;
         if (!lastMatch) return null;
         return { turn: parseInt(lastMatch[1]), player: lastMatch[2].trim() };
       }, playerName);
       expect(info).not.toBeNull();
       expect(info!.player).toBe(playerName);
       turnNumber = info!.turn;
     }, { timeout });
     return { turn: turnNumber };
     ```
   - This uses Playwright's `toPass` retry assertion which is preferred over manual polling loops
   - If `toPass` does not work in this Playwright version, fall back to `expect.poll()`:
     ```typescript
     await expect.poll(async () => {
       const info = await page.evaluate(/* same as above */);
       if (info && info.player === playerName) { turnNumber = info.turn; return true; }
       return false;
     }, { message: `Waiting for ${playerName}'s turn`, timeout, intervals: [200, 500, 1000] }).toBeTruthy();
     ```

4. `getPlayerName(page: Page): Promise<string>`

   - Fallback: Read from Angular scope via `page.evaluate()` if accessible
